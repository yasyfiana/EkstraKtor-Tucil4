Aplikasi Ekstraksi Informasi

Penggunaan :

  1. Masukan kata kunci dari informasi yang ingin dicari.
  2. Pilih file yang informasinya ingin diekstraksi.
  3. Pilih algoritma pencocokan string yang diinginkan
  4. Lalu klik tombol cek
  5. program akan menampilkan hasil ekstraksi

Tentang program :

PUSKUSI adalah aplikasi ekstraksi informasi. Proses ekstraksi yang dilakukan menggunakan pencocokan string dengan pilihan algoritma Booyer-Moore, Knuth-Morris-Pratt, regex. Dan proses ekstraksi waktu dan jumlah dilakukan dengan menggunakan regex.

Pogram ini dibuat untuk memenuhi tugas kecil matakuliah IF2211 Strategi Algoritma Tahun Ajaran 2019/2020 Teknik Informatika Institut Teknologi Bandung
